# CreditPathAI

CreditPathAI is a machine learning–based system designed to predict loan default risk and recommend personalized recovery actions.

## Objective
To automate loan recovery decision-making using data-driven insights.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- FastAPI (later)
- React.js (later)
