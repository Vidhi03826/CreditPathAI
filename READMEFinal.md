# CreditPathAI

Machine Learning Based Credit Risk Prediction System

## Overview

CreditPathAI is an end-to-end machine learning project that predicts loan repayment outcomes using historical loan data.

The project covers:

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Model Training
- Model Evaluation
- Flask Deployment
- Production Readiness Concepts

## Features

- Loan Repayment Prediction
- Feature Engineering Pipeline
- Logistic Regression Model
- Decision Tree Model
- Random Forest Model
- Model Comparison
- Feature Importance Analysis
- Confusion Matrix Visualization
- Flask Web Application
- Prediction Logging
- Health Monitoring Endpoint
